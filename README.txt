C'est un projet fait dans le cadre de la matière de programmation distribuée.


ATTENTION 
Pour utiliser le code il y a deux options :
- En localhost , il faut décommenter "lienMongo = "mongodb://localhost:27017/Prog-Dist-DB" dans DBconfig.js 
- Pour déployer dans des conteneurs le projet il faut décommenter "lienMongo = "mongodb://mongo:27017/Prog-Dist-DB"" dans DBconfig.js et lancer la commande : docker-compose up pour créer les conteneurs.
